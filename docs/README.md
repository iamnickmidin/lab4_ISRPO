# Geometric Lib

Библиотека для вычисления площадей и периметров геометрических фигур: круга, квадрата, прямоугольника и треугольника.

---

## Функции библиотеки

---

### Треугольник (`triangle.py`)

#### `area(a, h)`
Вычисляет площадь треугольника по основанию и высоте.

**Формула:**  
`S = (a * h) / 2`

**Аргументы:**
- `a` — основание треугольника (положительное число)  
- `h` — высота треугольника (положительное число)

**Возвращаемое значение:** площадь треугольника.

**Пример:**
area(10, 5) → 25.0

#### `perimeter(a, b, c)`
Вычисляет периметр треугольника.

**Формула:**  
`P = a + b + c`

**Аргументы:**
- `a`, `b`, `c` — длины сторон треугольника

**Пример:**
perimeter(3, 4, 5) → 12

---

### Квадрат (`square.py`)

#### `area(a)`
Вычисляет площадь квадрата.

**Формула:**  
`S = a²`

**Аргументы:**
- `a` — длина стороны квадрата

**Пример:**
area(5) → 25

#### `perimeter(a)`
Вычисляет периметр квадрата.

**Формула:**  
`P = 4a`

**Пример:**
perimeter(5) → 20

---

### Прямоугольник (`rectangle.py`)

#### `area(a, b)`
Вычисляет площадь прямоугольника.

**Формула:**  
`S = a * b`

**Аргументы:**
- `a` — первая сторона  
- `b` — вторая сторона  

**Пример:**
area(5, 3) → 15

#### `perimeter(a, b)`
Вычисляет периметр прямоугольника.

**Формула:**  
`P = 2(a + b)`

**Пример:**
perimeter(5, 3) → 16

---

### Круг (`circle.py`)

#### `area(r)`
Вычисляет площадь круга.

**Формула:**  
`S = π * r²`

**Аргументы:**
- `r` — радиус круга

**Пример:**
area(5) → 78.5398...

#### `perimeter(r)`
Вычисляет длину окружности.

**Формула:**  
`P = 2πr`

**Пример:**
perimeter(1) → 6.283

## Unit-тесты

Тесты находятся в каталоге `tests/` и написаны с использованием модуля `unittest`.

Каждый тест содержит описание:
- проверяемой функции  
- используемой формулы  
- принимаемых аргументов  
- ожидаемого результата  

### Запуск тестов

python3 -m unittest discover -s tests

---

## План тестирования

### Цели
- Проверить корректность вычисления площадей и периметров фигур.
- Убедиться, что формулы реализованы правильно.
- Подтвердить корректность работы при валидных входных данных.

### Область тестирования
Тестируются функции:
- `area()` всех фигур  
- `perimeter()` всех фигур  
- целые и дробные числа  

### Стратегия
- Модульное тестирование  
- Использование заранее рассчитанных эталонных значений  
- Тестирование корректных входных данных  

### Критерии успешного тестирования
- Все тесты проходят без ошибок  
- Результаты совпадают с математическими формулами  

### Ожидаемые результаты
- Подтверждение корректности вычислений  
- Отчёт с успешным прохождением тестов
